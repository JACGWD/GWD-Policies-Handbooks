# Web Sequence Committee

For each academic year, a committee made up by the teachers who teach in the GWD web sequence must be struck and given the mandate of guaranteeing the continued pertinence of the material taught in the web courses in GWD. This includes: 

- selection of target skills for graduates
- choice of software used in class
- maintaining balance of design vs development skills
- etc


The Web Sequence Committee should base parts of its conclusions based on the feedback from the [GWD Advisory Committee](./advisory-committee.md).
