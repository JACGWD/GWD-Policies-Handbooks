# Course Outline Committee: 

For each academic year, a **three person committee** must be struck and given the mandate of reviewing all course outlines for validity and conforming to established requirements.

## Course Outline Committee [IPESA Requirement]  

 Course outlines for new courses must be approved, before the courses begin, by the GWD department, then presented in a Program Committee meeting, then by Academic Council. For a start date of August, the approval date at Academic Council has to be April.
